ブログ記事の投稿方法

照株式会社 Webサイト — GitHub Pages でのブログ更新ガイド

全体の流れ

ブログ記事を投稿するには、以下の 2つのステップ だけです：

  1. 記事ファイルを作成して blog/ フォルダにアップロード
  2. blog.html の一覧に記事カードを追加

ステップ1：記事ファイルを作成する

1 TEMPLATE.html をコピー

GitHubのリポジトリで blog/TEMPLATE.html を開き、右上の鉛筆アイコン（Edit）をクリックして、全文をコピーします。

2 新しいファイルを作成

リポジトリのトップ → blog フォルダ → 「Add file」→「Create new file」をクリック。

ファイル名は以下のルールで付けてください：

YYYY-MM-DD-タイトル.html

例：

2026-04-01-sauna-trend.html
2026-04-15-kirinosato-renewal.html

3 テンプレートを貼り付けて、★マーク部分を編集

コピーしたテンプレートを貼り付け、以下の ★マーク が付いた箇所を書き換えます：

  • <title> 内の記事タイトル
  • <meta name="description"> の説明文
  • 日付（article-date）
  • カテゴリ（article-tag）— サウナ、メディア、お知らせ など
  • 記事タイトル（<h1>）
  • 記事本文（article-body 内）

4 本文の書き方

以下のHTMLタグだけ覚えれば記事が書けます：

<p>普通の段落はこのように書きます。</p>

<h2>大きな見出し</h2>

<h3>小さな見出し</h3>

<ul>
  <li>箇条書き1</li>
  <li>箇条書き2</li>
</ul>

<ol>
  <li>番号付きリスト1</li>
  <li>番号付きリスト2</li>
</ol>

<blockquote>
  引用文をここに書きます。
</blockquote>

<img src="../images/blog/ファイル名.jpg" alt="画像の説明">

<strong>太字にしたい部分</strong>

5 「Commit changes」で保存

画面下部の「Commit changes」をクリックすると、記事が公開されます。

POINT：画像を使いたい場合

images/blog/ フォルダを作って、そこに画像をアップロードしてください。記事からは <img src="../images/blog/ファイル名.jpg" alt="説明"> で表示できます。

ステップ2：ブログ一覧に記事カードを追加する

1 blog.html を編集

リポジトリのトップにある blog.html を開き、鉛筆アイコンで編集モードにします。

2 記事カードを追加

<!-- ▼ 記事カード --> のコメントの直後に、以下のコードをコピー＆貼り付けして、内容を書き換えます：

<a href="blog/YYYY-MM-DD-タイトル.html" class="blog-card fade-in">
  <div class="blog-card-thumb">
    <!-- サムネイル画像がある場合: <img src="images/blog/xxx.jpg" alt=""> -->
  </div>
  <div class="blog-card-body">
    <div class="blog-card-meta">
      <span class="blog-card-date">2026.04.01</span>
      <span class="blog-card-tag">サウナ</span>
    </div>
    <h3>ここに記事タイトル</h3>
    <p>ここに記事の要約を2〜3行で</p>
  </div>
</a>
注意

新しい記事は 一番上（既存の記事カードの上） に追加してください。そうすることで、新しい記事が一覧の先頭に表示されます。

3 「Commit changes」で保存

数分後にサイトに反映されます。

書き換える箇所のまとめ

毎回変更が必要なのは以下の部分だけです：

記事ファイル（blog/YYYY-MM-DD-xxx.html）

  • <title> — 記事タイトル
  • <meta name="description"> — 記事の説明
  • article-date — 日付
  • article-tag — カテゴリ
  • <h1> — 記事タイトル
  • article-body の中身 — 本文

一覧ページ（blog.html）

  • href — 記事ファイルへのリンク
  • 日付・カテゴリ・タイトル・要約
POINT

迷ったら、サンプル記事（blog/2026-03-19-sample.html）を見て参考にしてください。実際の記事がどのようなHTMLになっているか確認できます。
